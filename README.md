
# Flatten Array

Given a list of integer ordered radomly must flatten. 

For example for this input  

```Python
  [1, [2, [3, [4, 5]]]]
```
your code must return 
```Python
  [1, 2, 3, 4, 5] 
```


## Installation and Running

Install flatten_array

```bash
  git clone https://github.com/IvanBarron/flatten_array.git
  cd flatten_array
  python -m unittest
```
